# Demo Script

1. Open the dashboard and show current metrics.
2. Upload a fictional resume PDF.
3. Review the AI-generated structured fields and resume text preview.
4. Confirm the candidate into the formal list.
5. Log in as the department demo user and create an active JD from validated JSON.
6. Open the candidate detail page, select the JD, and run DeepSeek V4 Flash screening.
7. Show that every matched criterion contains an exact resume quote and that the stage is unchanged.
8. Send the assessment card to the configured WeCom group.
9. Confirm the Agent recommendation as the department user.
10. Show the changed stage, event audit trail, Tencent Docs columns, dashboard pass rate and screening duration.
