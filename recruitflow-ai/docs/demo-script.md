# Demo Script

1. Open the dashboard and show current metrics.
2. Upload a fictional resume PDF.
3. Review the AI-generated structured fields and resume text preview.
4. Confirm the candidate into the formal list.
5. Open the candidate detail page and preview the screening card.
6. Mark screening as passed.
7. Show the dashboard funnel update and event log.
8. Export the candidate CSV.
